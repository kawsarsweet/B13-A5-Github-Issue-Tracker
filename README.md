1️⃣ What is the difference between var, let, and const?

var, let, and const are used to declare variables in JavaScript, but they behave differently.

>>> var

1.It is function-scoped.
2.The same variable can be redeclared and updated.
3.It was mainly used in older versions of JavaScript.

>>>let

1.It is block-scoped (works inside { } blocks).
2.The same variable cannot be redeclared in the same scope.
3.The value can be updated later.

>>>const

1.It is also block-scoped.
2.The variable must be assigned a value when declared.
3.The value cannot be changed later.



2️⃣ What is the spread operator (...)?

The spread operator (...) is used to expand or spread elements of an array or object. It is commonly used to copy arrays or combine multiple arrays.


3️⃣ What is the difference between map(), filter(), and forEach()?

These three methods are used with arrays, but they serve different purposes.

>> map()

Executes a function on each element.
Returns a new array with modified values.

>> filter()

Returns elements that match a specific condition.
Returns a new array.


>>forEach()

Used to loop through array elements.
Does not return a new array.


4️⃣ What is an arrow function?

An arrow function is a shorter way to write functions in JavaScript, introduced in ES6. It makes the code simpler and cleaner.


5️⃣ What are template literals?

Template literals are used to create strings that include variables or expressions easily.They are written using backticks ( ) instead of quotes.
